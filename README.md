# Movie Box Office Data Wrangling & Analysis

## Overview
This collaborative data wrangling project analyzes movie box office performance using web scraping and data analysis techniques. The notebook extracts real-world film data, cleans and transforms it, and explores relationships between production budgets, revenues, genres, and other key film attributes.

## Project Objectives
- **Web Scraping**: Use Selenium to automatically collect movie data from online sources
- **Data Cleaning**: Use pandas to handle missing values, inconsistencies, and data quality issues
- **Exploratory Analysis**: Analyze relationships between film budgets, revenues, and genres
- **Visualizations**: Create insightful charts and summary statistics to identify trends

## Key Technologies
- **Python** - Core programming language
- **Selenium** - Web scraping and automated data collection
- **Pandas** - Data manipulation and cleaning
- **Matplotlib/Seaborn** - Data visualization
- **Jupyter Notebook** - Interactive development environment

## Dataset Features
The analysis explores:
- Production budgets vs. box office revenues
- Genre performance and profitability
- Relationships between film attributes and financial success
- Trends in movie industry data

## How to Use
1. Clone or download this repository
2. Install required packages: `pip install selenium pandas matplotlib seaborn jupyter`
3. Open the Jupyter Notebook: `jupyter notebook movie_box_office_analysis.ipynb`
4. Run all cells to see the data collection, cleaning, and analysis
